# Gemini Chatbot API

A modern, AI-powered English learning assistant built with Node.js and Google Gemini.

## Features
- **AI-Powered Corrections**: Uses Google Gemini Pro to correct grammar and vocabulary.
- **Modern UI**: Stylish glassmorphism design with animated backgrounds.
- **Real-time Interaction**: Fast and responsive chat interface.

## Tech Stack
- **Backend**: Express.js, Node.js
- **AI Integration**: Google Generative AI SDK
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. Run the server: `node index.js`
